# Tello_Control
相关信息见[主repo](https://github.com/zoeyuchao/tello_control/)  
这是初赛的一个完整实现

## 运行
基本的运行环境要求参考[主repo](https://github.com/zoeyuchao/tello_control/)  
因为彻底脱离了ros，直接运行
```cmd
python tello_state.py （直接从原版进行修改的，比较乱，而且图像刷新只会在指令传输完成后进行，但是基础功能已经完善了）
```
或者
```cmd
python tello_wrap.py
```
即可  
__Ctrl+C__ 终止后会自动降落
